# Hushmark Branding Policy

The Hushmark name, logo, icon, and related branding identify the original Hushmark project.

The Hushmark source code is licensed under GPL-3.0-or-later. This branding policy is separate from the source code license and does not limit rights granted by the GPL.

## Allowed Uses

You may use the Hushmark name to refer truthfully to the original project, for example:

- "based on Hushmark"
- "derived from Hushmark"
- "a fork of Hushmark"
- "packaged from the official Hushmark source"

Linux distributions and package maintainers should package and distribute Hushmark under the Hushmark name when the package is built from the official Hushmark source code, including reasonable downstream packaging patches, dependency patches, build-system patches, security fixes, and distribution integration changes.

## Not Allowed Without Permission

You may not use the Hushmark name, logo, icon, or related branding in a way that suggests that a modified version, fork, derivative product, or separately distributed product is the official Hushmark project or is endorsed by it, unless you have explicit permission.

You may not use the Hushmark name, logo, icon, or branding as the primary identity of a separate derivative product without explicit permission.

Modified versions distributed to end users should clearly identify themselves as modified, unofficial, downstream, or derived from Hushmark when there is a reasonable chance of user confusion.
